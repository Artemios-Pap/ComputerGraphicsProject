# Computer Graphics Project

This repository contains computer-graphics assignments and experiments. The first three parts (project-1A, project-1B, project-1C) are OpenGL-based C++ projects (with GLSL-style shaders). Later work is a Unity project (C#) located under project-2/Bob in Wonderland containing shader experiments, editor utilities, and demo scenes.

Short description
A mix of OpenGL C++ assignments (Parts 1–3) followed by a Unity C# project with shader experiments and real-time rendering demos.

Repository structure
- project-1A/ — Part 1 (OpenGL C++): Source-1A.cpp, SimpleVertexShader.vertexshader, SimpleFragmentShader.fragmentshader, ProjectVertexShader.vertexshader, ProjectFragmentShader.fragmentshader, and a readme.pdf.
- project-1B/ — Part 2 (OpenGL C++): Source-1B.cpp, P1BVertexShader.vertexshader, P1BFragmentShader.fragmentshader, and readme.pdf.
- project-1C/ — Part 3 (OpenGL C++): source-1C.cpp, P1C.vertexshader, P1C.fragmentshader, textures (texture-model-A.jpg, texture-model-B.jpg), and readme.pdf.
- project-2/Bob in Wonderland/ — Unity project (C#) with assets, shaders, and Unity package cache (Library/ may contain generated files).

Languages
- C# (Unity project)
- C++ (OpenGL assignments)
- GLSL/HLSL/ShaderLab (shader files)

Requirements
- OpenGL parts:
  - C++ compiler (GCC/Clang/MSVC) and OpenGL development headers/libraries.
  - Build/run instructions are provided in each part's readme or PDF (check readme.pdf or the DOCX file inside each part's folder).
- Unity parts:
  - Unity 2020.1 or later recommended.
  - .NET Standard 2.0 compatibility for editor/runtime code.

How to run
1. Clone the repository:
   git clone https://github.com/Artemios-Pap/ComputerGraphicsProject.git

2. OpenGL parts:
   - Enter a part folder (e.g., project-1A) and review Source-1A.cpp or Source-1B.cpp/source-1C.cpp for compiling instructions. These projects are plain C++ + OpenGL shaders; compile with your preferred toolchain.
   - If build scripts are missing, a typical build command for Linux/macOS with g++ might look like:
     g++ -std=c++17 -O2 Source-1A.cpp -lGL -lGLEW -lglfw -o part1
   - Ensure you have the required dependencies (GLFW/GLUT/GLEW or equivalent) installed.

3. Unity project:
   - Open the 'project-2/Bob in Wonderland' folder in Unity (recommended 2020.1+). If Library/ is present in the repo you can delete it locally — Unity will regenerate it.
